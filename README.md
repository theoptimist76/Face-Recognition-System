## A very Simple app for Face Recognition  

  - Collect Face Data
  - Build Face Classifier 
  - Recognize the face and display output

## Installation

1 : Install the requirements .

```sh
$ pip install -r  requirements.txt
```

2 : Run The App 

```sh
$ python app.py
```
